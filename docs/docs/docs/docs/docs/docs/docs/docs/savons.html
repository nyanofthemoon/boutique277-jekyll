Ingrédients Utilisés dans nos Savons

Tous nos ingrédients sont listés sur nos étiquettes selon la nomenclature internationale des ingrédients cosmétiques ou INCI.

Nous avons toujours de nouveaux produits en dévelopment! Par contre, les savons artisanaux sont classés en tant que “cosmétiques” par Santé Canada et doivent passer par un processus d’approbation avant d’arriver sur nos tablettes ou être vendus aux foires locales.
